Inclusão financeira é o acesso efetivo a serviços financeiros como:

- Crédito  
- Poupança  
- Pagamentos  
- Seguros  
- Previdência  
- Investimentos  

O acesso deve ser:
- Adequado às necessidades do usuário
- Acessível financeiramente
- Sustentável para o sistema
